1. **Clone or Download**
   ```bash

   git clone [repository-url]
   cd MomentumLabs
   
   # Or download and extract the ZIP file
   ```

2. **Open the Project**
   ```bash
   # Navigate to the project directory
   cd MomentumLabs
   
   # Open index.html in your browser
   open index.html
   # or
   start index.html
   # or simply double-click the file
   ```

### **Customization**

1. **Colors**: Modify CSS variables in `styles.css`
   ```css
   :root {
       --primary-bg: #111111;
       --accent-red: #E30613;
       /* ... other variables */
   }
   ```

2. **Content**: Update text and company information in `index.html`

3. **Portfolio Data**: Modify chart data in `script.js`
   ```javascript
   const portfolioData = {
   ```
